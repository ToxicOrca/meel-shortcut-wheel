Drop a 16x16 (and optionally 32x32) PNG named `tray-icon.png` here for the
system tray icon. If it is missing, Meel falls back to an empty placeholder
icon so the app still runs.

A dark, simple glyph (e.g. a ring / target ◎) reads best in both light and
dark Windows taskbars. Windows will scale automatically, but a crisp 32x32 is
recommended for high-DPI displays.
